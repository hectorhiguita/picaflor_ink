# components/

Reusable React components for Picaflor INK. Organized by domain:

- `ui/` — Primitive UI components (Button, Input, Badge, Card, etc.)
- `layout/` — Header, Footer, Navigation, WhatsApp CTA button
- `catalog/` — Product cards, gallery grid, filters, pagination
- `mockup/` — Mockup generator canvas and layer controls
- `cart/` — Cart drawer, cart item, cart summary
- `checkout/` — Checkout form, order summary, payment button
- `product/` — Product detail view and variant selector
- `designs/` — Design catalog and selector components
- `account/` — Login and registration UI
- `social/` — Social feed surface
- `brand/` — Brand-specific visual accents

Server Components are the default. Add `"use client"` only when interactivity
or browser APIs are required.
